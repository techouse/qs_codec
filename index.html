qs-codec¶

A query string encoding and decoding library for Python.

Ported from qs for JavaScript.

Usage¶

A simple usage example:

import qs_codec as qs

# Encoding
assert qs.encode({'a': 'b'}) == 'a=b'

# Decoding
assert qs.decode('a=b') == {'a': 'b'}
